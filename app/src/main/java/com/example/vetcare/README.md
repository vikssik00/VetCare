# VetCare

VetCare — мобильное Android-приложение для автоматизации работы ветеринарной клиники.  
Приложение предназначено для владельцев животных, ветеринаров и администратора клиники.

Проект разработан в рамках учебной работы.

---

## Описание проекта

VetCare позволяет владельцам животных вести учет своих питомцев, записываться на приём в ветеринарную клинику и просматривать историю посещений.  
Ветеринары работают с пациентами и медицинскими записями, а администратор управляет пользователями и расписанием клиники.

Приложение ориентировано на упрощение взаимодействия между клиентами клиники и персоналом, а также на централизованное хранение медицинской информации.

---

## Роли пользователей

### Владелец питомца
- Регистрация и вход в систему
- Просмотр и редактирование профиля
- Добавление и просмотр питомцев
- Просмотр медицинских записей
- Запись на приём
- Просмотр уведомлений

### Ветеринар
- Просмотр списка пациентов
- Просмотр и добавление медицинских записей
- Работа с расписанием приёмов
- Просмотр назначенных приёмов

### Администратор
- Управление пользователями (владельцы, ветеринары)
- Назначение ролей
- Управление расписанием клиники
- Контроль записей на приём

## Основной функционал

- Регистрация и авторизация пользователей (Firebase Authentication)

- Запись на приём к ветеринару
- Просмотр истории посещений
- Уведомления внутри приложения
- Административная панель

---

## Архитектура проекта

Проект построен с разделением логики:

- ui — экраны и фрагменты интерфейса
- models — модели данных
- adapters — адаптеры списков
- utils — вспомогательные классы и утилиты
- firebase — работа с Firebase

Реализовано разделение ролей пользователей и доступов к данным.

---

## Хранение данных

- Пользователи — Firebase Firestore (`users`)
- Питомцы — Firebase Firestore (`pets`)
- Приёмы — Firebase Firestore (`appointments`)
- Медицинские записи — Firebase Firestore (`medical_records`)

Данные синхронизируются с сервером Firebase.
